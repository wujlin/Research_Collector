## Abstract

SocialJax is a JAX-based suite of sequential social dilemma environments and algorithms. It targets the computational bottleneck of Melting Pot-style MARL evaluation and reports large speedups while preserving social-dilemma structure.

## Why It Matters

This is the practical continuation of the Leibo benchmark line. It matters if you later want to run many population-level simulations rather than only reason about them conceptually.

## Connection

For route generation and agent-based urban simulation, SocialJax is a reminder that scalable environment implementation can be as important as model design.
