## Research Roles

- `Baronchelli classic convention line`: local interaction -> global convention -> consensus/tipping.
- `Baronchelli mobility line`: repeated location set as a conserved individual-level structure.
- `Leibo MARL line`: social dilemma as temporally extended policy learning, not one-shot game choice.
- `Leibo benchmark line`: evaluate social generalization across populations, partners, and scenarios.
- `Generative agents line`: LLM-based agent societies need population-level and interactionist analysis.
