# Knowledge Cards

这个目录用于存放需要反复回忆的核心概念、推导和公式卡片。

约定：

- 每张卡片只讲一个核心对象。
- 顺序固定为：问题 -> 定义 -> 推导 -> 结论 -> 常见混淆。
- 避免把不同层面的对象混在一起。
- 公式统一使用正常数学模式，避免伪公式写法。
